# Atom Shortcut Key

- Show / Hide navigation left side bar
  ```
  Ctrl \
  ```
- Comment out
  ```
  Ctrl /
  ```

- Split left screen
  ```
  (Ctrl k) + leftbutton
  ```
  
- Split right screen
  ```
  (Ctrl k) + rightbutton
  ```
  
- Indent 
  ```
  Tab with cursor at the left end the line
  or
  Ctrl ] with cursor at any place on the line
  ```
- Unindent
  ```
  Shift Tab or Ctrl [
  ```
- Duplicate
  ```
  Ctrl Shift d
  ```
- Select 1 word
  ```
  Ctrl d
  ```
- Select next same word
  ```
  Ctrl d
  ```
- Select all same word
  ```
  Shift F3
  ```
- Select a line
  ```
  Ctrl l
  ```
- Delete 1 line
  ```
  Shift Delete
  ```
