# Visual Studio Code Useful Shorcut

## Display
- Show / Hide navigation left side bar
  ```
  Ctrl + B
  ```
- Split screen
  ```
  View / Editor Layout
  ```
- Split screen Vertical
  ```
  Ctrl + \
  ```
- Close windows
  ```
  Ctrl + K W
  ```
- Close all windows
  ```
  Ctrl + K U
  ```
## Basic Editing
- Copy line
  ```
  Ctrl + C
  ```
- Cut line
  ```
  Ctrl + X
  ```
- Delete 1 line
  ```
  Shift + Delete
  ```
- Scroll line up/down
  ```
  Alt + ↑ / ↓ 
  ```
- Scroll page 1 line up/down - Cursor stay the same location
  ```
  Ctrl + ↑ / ↓ 
  ```
- Insert line below
  ```
  Enter or Ctrl + Enter
  ```
- Insert line above
  ```
  Ctrl + Shift + Enter
  ```
- Jump to matching bracket
  ```
  Ctrl + Shift + \
  ```
- Comment out
  ```
  Ctrl + /
  ```
- Indent
  ```
  Tab with cursor at the left end the line
  or
  Ctrl + ] with cursor at any place on the line
  ```
- Outdent
  ```
  Shift Tab or Ctrl [
  ```
- Duplicate
  - Keep cursor on top element
    ```
    Alt Shift UpArrow
    ```
  - Keep cursor on bottom element
    ```
    Alt Shif DownArrow
    ```
- Select current word
  ```
  Ctrl D
  ```
- Select next current word
  ```
  Ctrl D
  ```
- Select all current words
  ```
  Ctrl Shift L
  ```
- Select line
  ```
  Ctrl I
  ```
- MultiCursor
  ```
  Alt Click
  ```

html:5 + Enter or html: (see the select html:5) + Tab
